# 📖 Overview

SkillForge LMS is a complete Learning Management System developed as a Full Stack project. The platform allows students to register, browse premium courses, enroll through an integrated payment flow, track lesson progress, and unlock certificates after successful course completion.

The application follows a modular backend architecture with REST APIs, MongoDB database integration, JWT authentication, and a responsive frontend interface.
